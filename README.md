## À propos du projet

Cette branche (`main`) est dédiée au développement continu du système expert. Elle contient l'intégralité du code source, y compris les outils de conception et les historiques de prompts IA.

### Objectifs de développement :
* **Évolutions** : Ajout de nouvelles règles cliniques et affinement du moteur.
* **Refactorisation** : Amélioration de la structure Lisp et des performances.
* **Tests Avancés** : Mise en place de suites de tests automatisées.

## Structure du projet

* `code-source/` : Contient le moteur Lisp et les fichiers de prompts (`prompt-*.txt`).
* `tests/` : (À venir) Tests unitaires et d'intégration.
* `rapport/` & `presentation/` : Sources des documents académiques.

## Roadmap

- [ ] Intégration de facteurs de certitude (logique floue).
- [ ] Interface graphique (GUI) pour la saisie des données patient.
- [ ] Export des diagnostics au format JSON/PDF.
- [ ] Support multi-langue pour les explications du moteur.
